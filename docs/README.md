## 사다리 게임 기능 목록

* `Couch` : 코치
    * [] : 코치의 이름이 2 ~ 4 사이인지 확인한다.
    * [] : 못 먹는 음식이 2개 이하인지 확인한다.

* `Recommendation` : 추천
    * [] : 추천 메뉴를 제공한다.
      * [] : 같은 카테고리가 2개 까지인지 확인한다.
      * [] : 중복 메뉴가 없는지 확인한다.
      * [] : 추천 메뉴에 못 먹는 메뉴가 없는지 확인한다.

* `Category` : 카테고리
    * [] : 카테고리 별로 음식을 저장하고 있다.
